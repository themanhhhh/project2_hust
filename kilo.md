# UI reference rules

## Goals
- Chỉ học theo bố cục, khoảng cách, nhịp thị giác và cấu trúc section
- Không sao chép nguyên văn text, logo, ảnh thương hiệu

## Frontend stack
- React + Tailwind
- Component hóa theo section
- Responsive mobile/tablet/desktop

## Output style
- Mỗi lần chỉ sửa 1 section lớn
- Trước khi code, mô tả ngắn layout sẽ làm
- Sau khi code, nêu file nào đã thay đổi
